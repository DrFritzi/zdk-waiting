zdk-waiting

the "zdk-waiting" component emulates the loading bar which is used in android applications

The component is based on the use of the polymer livrary

to use it, create in your html file body :


    <zdk-waiting></zdk-waiting>

and here is the result :

The component exposes some attributes :

  • Color
  • backColor
  • segment

Here an example changing the bar color :


    <zdk-waiting color="Tomato"></zdk-waiting>

another example changing the segment attribute and Color :


    <zdk-waiting segment="10" color="#00cc66"></zdk-waiting>

The component could be manipulated programmatically with javascript


    var bar = document.createElement("zdk-waiting");
    bar.setAttribute("color","DarkOrange");
    bar.setAttribute("backcolor","#eeeeee");
    document.body.appendChild(bar);


    <label for="color">Choose a color for The bar :</label>
    <input id="color" type="color" value="#00cc66"/>
    <label for="segment">Choose number of segment : (<span id="segmentValue">5</span>)</label>
    <input id="segment" type="range" min="1" max="20" value="5"/>
    <zdk-waiting id="wait" color="#00cc66" segment="5"></zdk-waiting>
    <script>
    var wait = document.querySelector("zdk-waiting#wait");
    var color = document.querySelector("#color");
    color.addEventListener("change", function () {
        wait.setAttribute("color", color.value);
    }, false);
    var segment = document.querySelector("#segment");
        segment.addEventListener("change", function () {
        document.querySelector("#segmentValue").textContent = segment.value;
        wait.setAttribute("segment", segment.value);
    });
    </script>
